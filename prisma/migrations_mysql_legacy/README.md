# Legacy MariaDB Migrations

Folder ini menyimpan riwayat migration lama yang dibuat untuk provider MySQL/MariaDB.

Migration ini tidak dijalankan oleh Prisma setelah DRSIS berpindah ke PostgreSQL. Folder aktif Prisma sekarang adalah `prisma/migrations`, dengan baseline PostgreSQL baru.

Gunakan isi folder ini hanya sebagai referensi audit atau kompatibilitas legacy saat membaca instalasi lama.
